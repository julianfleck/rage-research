# RAGE — Project Context

**Full name:** Recursive Attention-Guided Exploration (earlier: Recursive, Agentic Graph Embeddings)
**Status:** Active research + implementation
**Period:** 2024–present
**Relation to Recurse:** RAGE is the substrate architecture; Recurse is the commercial product built on it

## Scope

RAGE investigates the **dynamic properties of knowledge substrates**: how meaning, attention, and structure evolve through interaction. Core concerns:

- Frame semantics as recursive knowledge primitives
- Convergence/divergence dynamics in retrieval
- Phase mechanics (coupling, oscillation, decay) as substrate properties
- Write-back: structure that learns from reasoning
- Substrate observability for multi-agent coordination

## Distinction from Recurse

| RAGE | Recurse |
|------|---------|
| Research program + architecture | Commercial product |
| Dynamic substrate properties | User-facing knowledge graph |
| Frame semantics, phase mechanics | Scopes, proxy, persistence |
